# Problem: Signal Delivery
 
문제 요약: 격자에서 본사 S가 보낸 신호가 지사 E에 도착하는 데 걸리는 최소 시간을 구하라.

## 풀이

### 문제 정보
- 난이도: Lv 03 정도 (기준은 프로그래머스)
- 주제: 최단 경로, 다익스트라, 격자 그래프, 방향 상태
- 문제 핵심 
  - 위치와 진행 방향을 함께 상태로 관리
  - 안테나와 본사에서만 방향 전환 가능
  - 저항력을 이동 비용으로 다익스트라 수행
- 최초 풀이: 260630 
- 수정 버전: 260712 

### 접근법
1. Solution1: 방향 상태 기반 다익스트라 
   - 현재 위치와 진행 방향을 하나의 상태로 두고, 칸의 종류에 따라 방향 유지 또는 방향 전환을 하며 최소 시간을 갱신한다. 


## Solution1: 방식 (권장)
### 코드 설계
1. N, M과 격자 정보를 입력받는다.
2. S와 E의 위치를 찾는다.
3. `dist[r][c][dir]` 배열을 만든다.
   - `(r, c)` 칸에 `dir` 방향으로 도착했을 때의 최소 시간이다.
4. 시작점 S에서는 여덟 방향 중 하나를 선택할 수 있으므로, 모든 방향 상태를 비용 0으로 우선순위 큐에 넣는다.
5. 우선순위 큐에서 현재 상태를 꺼낸다. 
   - 현재 칸이 E이면 현재 비용을 출력한다. 
   - 현재 칸이 S 또는 .이면 여덟 방향으로 이동을 시도한다. 
   - 현재 칸이 숫자라면 기존 진행 방향으로만 이동한다. 
6. 이동 가능한 다음 칸에 대해 현재 칸의 저항력을 더해 최소 시간을 갱신한다. 
7. 끝까지 E에 도착하지 못하면 -1을 출력한다. 

### 코드 구현
- 핵심 포인트
- 시간복잡도: `O(NM log(NM))` 
- 공간복잡도: `O(NM)`
