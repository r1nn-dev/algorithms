# Problem: Racing Track 
 
문제 요약: 순환하는 격자 도로에서 자동차가 (1, 1)에서 오른쪽으로 출발할 때, T초 후 위치를 구하라. 

## 풀이

### 문제 정보
- 난이도: Lv 02 정도 (기준은 프로그래머스)
- 주제: 그래프 탐색, 사이클 추적, 격자 이동, 모듈러 연산
- 문제 핵심 
  - 순환 경로를 실제 이동 순서대로 저장 
  - `T % 경로 길이`로 최종 위치 계산 
- 최초 풀이: 260626 
- 수정 버전: 260707 

### 접근법
1. Solution1: 사이클 경로 직접 추적 + 모듈러 연산
   - 도로 전체가 하나의 순환 경로이므로, 시작점부터 자동차가 실제로 이동하는 순서대로 도로 칸을 저장한다.
   - `T`가 최대 `10^18`이므로 직접 `T`초를 시뮬레이션하면 안 된다.
   - 대신 한 바퀴 경로 길이를 `K`라고 하면, `T % K`번째 위치가 정답이다.


## Solution1: 사이클 경로 직접 추적 + 모듈러 연산
### 코드 설계
1. 입력으로 N, M, T, 그리고 트랙 정보를 받는다.
2. 자동차의 시작 위치 (0, 0)을 경로 리스트에 저장한다.
3. 자동차는 처음에 오른쪽으로 이동하므로 다음 위치는 (0, 1)로 시작한다.
4. 현재 칸에서 인접한 도로 칸 중 직전에 온 칸이 아닌 칸으로 계속 이동한다.
5. 다시 시작점 (0, 0)에 도착하면 한 바퀴 경로 추적을 종료한다.
6. `T % 경로길이`에 해당하는 칸을 찾는다.
7. 문제는 1-based 행, 열을 요구하므로 각각 +1 해서 출력한다.

### 코드 구현
- 핵심 포인트
- 시간복잡도: `O(NM)`
- 공간복잡도: `O(NM)`
