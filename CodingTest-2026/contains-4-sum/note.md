# Problem — Sum of Numbers Containing 4

### 문제
```
수가 N개 주어질 때, 숫자 4가 포함된 정수인 수의 합을 구하라.
```
### 입력
첫째 줄에 정수 N이 주어진다.   
둘째 줄부터 N개의 줄에 걸쳐 i번째 줄에 i번째 수 m<sub>i</sub> 가 주어진다.
- 1 ≤ N ≤ 10,000
- 1 ≤ m<sub>i</sub> ≤ 100,000
- 입력으로 주어지는 모든 수는 정수이다.
### 출력
- 조건을 만족하는 정수인 수의 합을 출력한다.

```
예시 1:
입력: 
6
1
2
3
4
8
14
출력: 18

예시 2:
입력:
3
4
14
40
출력: 58
```


## 풀이

### 주제
- 기본 알고리즘, 문자열 변환, 조건 누적 합
- 핵심:
  - 정수 입력 처리
  - 숫자를 문자열로 변환
  - 특정 문자 포함 여부 확인
  - 조건을 만족하는 값만 합산

### 문제 이해
- 입력: 첫째 줄에 정수의 개수 N이 주어진다.
- 둘째 줄부터 N개의 줄에 걸쳐 정수 m<sub>i</sub>가 주어진다.
- 각 정수에 숫자 `4`가 포함되어 있는지 확인해야 한다.
- 숫자 `4`가 포함된 정수만 합계에 더한다.
- 최종적으로 조건을 만족하는 정수들의 합을 출력한다.

### 접근법
1. 숫자를 문자열로 변환해서 확인
  - 정수 자체는 각 자리 숫자를 바로 확인하기 어렵다.
  - `String.valueOf(number)`를 사용하면 숫자를 문자열로 바꿀 수 있다.
  - 문자열에서 `contains("4")`를 사용하면 숫자 4가 포함되어 있는지 쉽게 확인할 수 있다.

2. 입력받자마자 바로 검사
  - 모든 숫자를 배열에 저장할 필요는 없다.
  - 각 숫자는 한 번만 사용되므로 입력받는 즉시 4 포함 여부를 검사하고, 조건을 만족하면 합계에 더하면 된다.

### 코드 설계
1. `BufferedReader`로 입력을 받는다.
2. 첫 번째 줄에서 N을 입력받는다.
3. 합계를 저장할 변수 `sum`을 0으로 초기화한다.
4. N번 반복하면서:
  - 정수 `number`를 입력받는다.
  - `String.valueOf(number)`로 문자열로 변환한다.
  - 변환된 문자열에 `"4"`가 포함되어 있는지 확인한다.
  - 포함되어 있으면 `sum`에 `number`를 더한다.
5. 반복이 끝나면 `sum`을 출력한다.


## 코드 구현
### 방법1: 문자열 변환 방식 (권장)
- 핵심 포인트
  - `Integer.parseInt()`로 입력 문자열을 정수로 변환한다.
  - `String.valueOf(number)`로 정수를 문자열로 변환한다.
  - `contains("4")`로 숫자 4가 포함되어 있는지 확인한다.
  - 조건을 만족하는 숫자만 `sum`에 더한다.
  - 배열에 저장하지 않고 입력받는 즉시 처리하면 메모리를 더 적게 사용한다.
- 시간복잡도: O(N × K)
  - N은 입력되는 숫자의 개수이다.
  - K는 각 숫자의 자릿수이다.
  - m<sub>i</sub> ≤ 100,000이므로 K는 매우 작다.
- 공간복잡도: O(1)
  - 배열을 사용하지 않고 입력받은 숫자를 바로 처리한다.

### 방법2: 산술 연산 방식
- 핵심 포인트
  - 문자열로 변환하지 않고 % 10, / 10을 이용해 각 자리 숫자를 확인한다.
  - number % 10으로 마지막 자리 숫자를 확인한다.
  - number / 10으로 마지막 자리 숫자를 제거한다.
  - 어떤 자리에서라도 4가 나오면 해당 숫자는 합산 대상이다.
  - 문자열 변환 없이 처리할 수 있지만, 코드가 조금 더 길어진다.
- 시간복잡도: O(N × K)
  - 각 숫자의 자릿수를 한 자리씩 확인한다.
- 공간복잡도: O(1)
  - 추가 배열 없이 변수만 사용한다.
