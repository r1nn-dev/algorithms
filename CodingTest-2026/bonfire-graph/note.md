# Problem: Bonfire Graph
 
문제 요약: 가중치가 있는 달집 그래프에서 하나의 달집에 불을 붙였을 때, 모든 달집이 타는 데 걸리는 시간의 최솟값을 구하라.

## 풀이

### 문제 정보
- 난이도: Lv 03 정도 (기준은 프로그래머스)
- 주제: 트리, 그래프 탐색, 트리 지름, 트리 반지름
- 문제 핵심 
  - 트리의 지름 양 끝점 찾기
  - 각 정점의 최대 거리 중 최솟값 계산
- 최초 풀이: 260630 
- 수정 버전: 260712 

### 접근법
1. Solution1: 트리 지름 기반 반지름 계산 
   - 트리 지름의 양 끝점에서 각 정점까지의 거리를 구한 뒤, 각 정점의 최대 거리 중 최솟값을 정답으로 구한다.


## Solution1: 트리 지름 기반 반지름 계산 
### 코드 설계
1. N을 입력받고 인접 리스트를 만든다.
2. `i = 1`부터 `N - 1`까지 입력을 받는다.
   - `A_i`와 `i + 1`번 달집을 길이 `L_i`로 양방향 연결한다.
3. 임의의 정점 1에서 가장 먼 정점 start를 찾는다.
4. `start`에서 가장 먼 정점 `end`를 찾는다.
   - 이때 `start`와 `end`가 트리 지름의 양 끝점이다. 
5. `start`에서 모든 정점까지의 거리 배열 `distStart`를 구한다.
6. `end`에서 모든 정점까지의 거리 배열 `distEnd`를 구한다.
7. 각 정점 i에 대해 `max(distStart[i], distEnd[i])`를 계산한다.
8. 반환 결과: 그중 최솟값을 출력한다. 

### 코드 구현
- 핵심 포인트
- 시간복잡도: `O(N)` 
- 공간복잡도: `O(N)`
