# 문제 — Coprime Pairs Sum N

### 문제
```
기약분수는 분자와 분모의 최대공약수가 1인 분수를 의미한다.
정수 N이 주어질 때, 다음 조건을 만족하는 두 정수 a,b의 쌍의 개수를 구하시오.
- a + b = N
- 1 ≤ a ≤ b ≤ N
- gcd(a, b) = 1
위 조건을 만족하는 (a, b) 쌍의 개수를 출력하라.
```

### 입력
첫째 줄에 정수 \( N \)이 주어진다.
- 1 ≤ N ≤ 10<sup>7</sup>

### 출력
- 조건을 만족하는 정수 쌍의 개수를 출력한다.

```
예시 1:

입력: 3
출력: 1

예시 2:

입력: 5
출력: 2
```
