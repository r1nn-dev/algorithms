# Problem: Dice Grid Travel
 
문제 요약: 격자에서 주사위 눈만큼 오른쪽 또는 아래로 이동해 목적지까지 가는 경우의 수를 구하라.

## 풀이

### 문제 정보
- 난이도: Lv 02 정도 (기준은 프로그래머스)
- 주제: DP, 격자 이동
- 문제 핵심 
  - `dp[r][c]` 사용
  - 위/왼쪽 6칸 확인
  - 휴식 칸은 0 처리
- 최초 풀이: 260527 
- 수정 버전: 260705 

### 접근법
1. Solution1: DP 격자 이동 방식 
   - 현재 칸으로 올 수 있는 위치는 위쪽 1~6칸, 왼쪽 1~6칸뿐이므로 DP로 누적하면 된다. 


## Solution1: 방식 (권장)
### 코드 설계
1. N, M, K를 입력받는다.
2. `rest[r][c]` 배열에 휴식 칸 위치를 표시한다.
3. `dp[1][1] = 1`로 출발지를 초기화한다.
4. 모든 칸을 순회하면서 현재 칸이 휴식 칸이면 건너뛴다. 
   - 위쪽 1~6칸에서 오는 경우를 더한다.
   - 왼쪽 1~6칸에서 오는 경우를 더한다.
   - 값을 1_000_000_007로 나눈 나머지로 저장한다.
5. 반환 형식: `dp[N][M]`을 출력한다.

### 코드 구현
- 핵심 포인트
- 시간복잡도: `O(N * M * 6)`
- 공간복잡도: `O(N * M)`
