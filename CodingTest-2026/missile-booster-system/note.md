# Problem: Missile Booster System 
 
문제 요약: 부스터를 한 번 사용할 짝수 시각을 골라 미사일들의 총 비행 시간을 최소화한다. 

## 풀이

### 문제 정보
- 난이도: Lv 03 정도 (기준은 프로그래머스)
- 주제: 스위핑, 구간 겹침, 정렬, 누적합 
- 문제 핵심 
  - 비행 구간 `[start, end)`
  - 겹치는 미사일 수 계산
  - `long` 범위 처리
- 최초 풀이: 260527 
- 수정 버전: 260705

### 접근법
1. Solution1: 스위핑 + 구간 겹침 계산
   - 부스터는 현재 날아가는 미사일마다 비행 시간을 1씩 줄이므로, 동시에 날아가는 미사일 수가 가장 많은 시각을 찾으면 된다.


## Solution1: 스위핑 + 구간 겹침 계산 
### 코드 설계
1. 미사일 개수 N을 입력받는다.
2. 각 미사일마다 x, y, start를 입력받는다.
3. 거리의 제곱 `distSquared = x * x + y * y`를 계산한다.
   - 기본 비행 시간은 `2 * distSquared`
   - 비행 구간은 `[start, start + flightTime)`
4. 시작 시각에는 `+1`, 도착 시각에는 `-1` 이벤트를 저장한다.
5. 이벤트를 시간순으로 정렬한 뒤 현재 날아가는 미사일 수의 최댓값을 구한다.
6. 반환 결과: `전체 기본 비행 시간 - 최대 겹침 수`를 출력한다.

### 코드 구현
- 핵심 포인트
- 시간복잡도: 'O(N log N)'
- 공간복잡도: 'O(N)'
