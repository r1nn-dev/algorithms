# Problem: Island Perimeter

### 문제
문제: 
```text
0과 1로 이루어진 2차원 정수 배열 grid가 주어집니다.
grid[i][j]가 1이면 해당 칸은 육지를 의미하고, 0이면 물을 의미합니다.  
육지는 상하좌우로 인접한 경우에만 서로 연결된 것으로 봅니다.
대각선 방향으로 맞닿아 있는 칸은 연결된 것으로 보지 않습니다.

각 칸은 한 변의 길이가 1인 정사각형이며, 격자 바깥은 모두 물로 둘러싸여 있다고 가정합니다.

grid에 표시된 하나의 섬에 대해, 섬의 둘레 길이를 return 하도록 solution 함수를 완성해주세요.
```
- 제한사항
  - 1 <= grid.length <= 100
  - 1 <= grid[i].length <= 100
  - grid[i][j]는 0 또는 1이다.
  - grid에는 하나 이상의 육지 칸이 존재한다.
  - 모든 육지 칸은 상하좌우 방향으로 연결되어 하나의 섬을 이룬다.
  - 섬 내부에는 물로 이루어진 호수가 없다.
  - 대각선 방향으로 인접한 육지는 연결된 것으로 보지 않는다.

### 입력
- 0과 1로 이루어진 2차원 정수 배열 `grid`
### 출력 
- 섬의 둘레 길이를 나타내는 정수
### 입출력 예시
```
예시 1:
입력:
grid = [[0,1,0],[1,1,1],[0,1,0]]
출력: 12

예시 2:
입력:
grid = [[1]]
출력: 4
```
- 예시 1: 십자가 모양의 섬이 만들어진다. 육지 칸은 총 5개이므로 섬의 바깥쪽 둘레는 12이다.
- 예시 2: 육지 칸이 1개이므로 섬의 둘레는 4이다.

- 문제 핵심:
  - 각 육지 칸마다 상하좌우 확인
  - 물 또는 경계일 때만 둘레에 카운트
- 최초 풀이: 260427

### 문제 이해
- 입력: 0(물)과 1(육지)로 이루어진 2차원 정수 배열 grid
- 해야 할 일:
  - 모든 육지 칸을 찾는다.
  - 각 육지 칸의 상하좌우를 확인한다.
  - 해당 방향이 격자 바깥이거나 물이면 둘레에 포함한다.
  - 해당 방향이 육지이면 둘레에 포함하지 않는다. 
- 주의할 점:
  - 대각선 방향은 연결로 보지 않는다.
  - 격자 바깥은 모두 물로 취급한다.
- 반환 형식: 섬의 둘레 길이를 int형으로 반환한다.

### 접근법
1. 육지 칸 기준 4방향 확인 방식 (권장)
   - 둘레는 육지 칸의 네 방향 중 물 또는 격자 바깥과 맞닿은 변의 개수이므로, 모든 육지 칸에서 4방향을 검사하면 된다.

### 코드 설계
1. grid의 행 개수와 열 개수를 변수에 저장한다.
2. 초기화할 변수
   - `int rows = grid.length`: 전체 행 개수
   - `int cols = grid[0].length`: 전체 열 개수
   - `int perimeter = 0`: 섬의 둘레 길이를 누적할 변수 
   - `int[] dr = {-1, 1, 0, 0}`: 상하좌우 방향 중 행 이동값 
   - `int[] dc = {0, 0, -1, 1}`: 상하좌우 방향 중 열 이동값 
3. 반복문 안에서 할 일
   - 모든 칸을 행 기준, 열 기준으로 순회한다.
   - 현재 칸이 물이면 건너뛴다.
   - 현재 칸이 육지이면 상하좌우 4방향을 확인한다.
   - 조건에 따른 갱신
     - 다음 위치가 격자 바깥이면 perimeter++ 한다.
     - 다음 위치가 물이면 perimeter++ 한다.
     - 다음 위치가 육지이면 아무것도 더하지 않는다.
   - 누적: 육지 칸이 물 또는 격자 바깥과 맞닿은 변의 개수만큼 perimeter에 누적한다.
   - 비교
     - `nextRow < 0`, `nextRow >= rows`, `nextCol < 0`, `nextCol >= cols`로 격자 범위를 벗어나는지 확인한다.
     - 범위 안에 있다면 grid[nextRow][nextCol] == 0인지 확인한다. 
4. 반환 형식: 계산된 섬의 둘레 길이 perimeter를 int형으로 반환한다.

## 코드 구현

### Solution1: 육지 칸 기준 4방향 확인 방식 (권장)
- 시간복잡도: O(R * C)
  - R은 행 개수, C는 열 개수이다.
  - 모든 칸을 한 번씩 확인하고, 육지 칸마다 최대 4방향만 검사한다.
- 공간복잡도: O(1)
  - 방향 배열 외에 추가 자료구조를 사용하지 않는다.
