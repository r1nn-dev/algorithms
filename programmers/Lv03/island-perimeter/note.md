# Problem: Island Perimeter

### 문제
문제: 
```text
0과 1로 이루어진 2차원 정수 배열 grid가 주어집니다.
grid[i][j]가 1이면 해당 칸은 육지를 의미하고, 0이면 물을 의미합니다.  
육지는 상하좌우로 인접한 경우에만 서로 연결된 것으로 봅니다.
대각선 방향으로 맞닿아 있는 칸은 연결된 것으로 보지 않습니다.

각 칸은 한 변의 길이가 1인 정사각형이며, 격자 바깥은 모두 물로 둘러싸여 있다고 가정합니다.

grid에 표시된 하나의 섬에 대해, 섬의 둘레 길이를 return 하도록 solution 함수를 완성해주세요.
```
- 제한사항
  - 1 <= grid.length <= 100`
  - 1 <= grid[i].length <= 100
  - grid[i][j]는 0 또는 1이다.
  - grid에는 하나 이상의 육지 칸이 존재한다.
  - 모든 육지 칸은 상하좌우 방향으로 연결되어 하나의 섬을 이룬다.
  - 섬 내부에는 물로 이루어진 호수가 없다.
  - 대각선 방향으로 인접한 육지는 연결된 것으로 보지 않는다.
