# Problem: Sum of Digits
  
### 문제 
문제: 정수 `n`이 매개변수로 주어질 때, `n`의 각 자리 숫자의 합을 return하도록 solution 함수를 완성하라.
- 제한사항: 0 ≤ n ≤ 1,000,000

### 입력
- 정수 `n`
### 출력
- `n`의 각 자리 숫자의 합을 반환한다.
### 입출력 예시
```
예시 1:
입력: 930211
출력: 16
```
- `9 + 3 + 0 + 2 + 1 + 1 = 16`을 return한다.


## 풀이

### 문제 정보
- 플랫폼: 프로그래머스
- 문제 링크: https://school.programmers.co.kr/learn/courses/30/lessons/120906
- 난이도: Lv 01 
- 주제: 기본 연산, 문자열 변환, 자릿수 추출
- 문제 핵심:
  - 숫자를 문자열로 변환하거나, 나머지 연산으로 각 자릿수 추출
- 최초 풀이: 260406
- 수정 버전: 264027

### 접근법
1. 방법1: 문자열 변환 방식
   - `String.valueOf(n)`으로 정수를 문자열로 변환한다.
   - `toCharArray()`로 각 문자를 순회한다.
   - 문자 숫자는 `ch - '0'`을 통해 실제 정수 값으로 변환한다.
   - 코드가 직관적이므로 권장 풀이로 사용한다.

2. 방법2: 나머지 연산 방식
   - `n % 10`으로 일의 자리 숫자를 추출한다.
   - `n /= 10`으로 일의 자리를 제거한다.
   - 숫자 자체를 직접 처리하므로 문자열 변환이 필요 없다.
   - 단, `n = 0`인 경우 while문이 실행되지 않지만 결과는 그대로 `0`이므로 정상 처리된다.

### String 변환
1. `charAt(i) - '0’` : ASCII. 인덱스로 문자에 접근한 뒤, 문자 코드값 차이를 이용해 정수로 변환
2. `toCharArray()` + `ch - '0'`  : 문자 배열로 변환 후, for문으로 순회하면서 정수로 변환 
3. `Character.getNumericValue()` : 의미가 명확한 표준 메서드
4. `Integer.parseInt(String.valueOf(ch))` : char → String → int 이중 변환

### 코드 설계 - 코드 구현 순서
1. 합계를 저장할 변수 `sum`을 0으로 초기화한다.
2. `String.valueOf(n)`으로 정수 `n`을 문자열로 변환한다.
3. 변환된 문자열을 문자 배열로 바꿔 한 글자씩 순회한다.
4. 각 문자 `ch`를 `ch - '0'`으로 정수로 변환한다.
5. 변환한 값을 `sum`에 더한다.
6. 반복이 끝나면 `sum`을 반환한다.

## 코드 구현

### Solution1: 문자열 변환 방식 (권장)
- 핵심 포인트
  - `String.valueOf(n)`으로 정수를 문자열로 변환
  - `toCharArray()`로 문자 배열 순회
  - `ch - '0'`으로 문자 숫자를 정수로 변환
- 시간복잡도: O(d), d = 자릿수
- 공간복잡도: O(d)
  - 문자열과 문자 배열을 사용한다.

### Solution2: 나머지 연산 방식
- 핵심 포인트
  - `n % 10`으로 일의 자리 추출
  - `n /= 10`으로 일의 자리 제거
- 시간복잡도: O(d), d = 자릿수
- 공간복잡도: O(1)
