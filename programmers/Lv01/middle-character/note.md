# Problem: Get Middle Character

### 문제
문제: 단어 s의 가운데 글자를 반환하세요. (짝수면 가운데 2개)
```text
단어 s의 가운데 글자를 반환하는 함수, solution을 만들어 보세요.
단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.
```
- 제한사항
  - s는 길이가 1 이상, 100이하인 스트링.

### 입력
- 문자열 s
### 출력 
- 문자열 s의 가운데 글자
- 문자열 길이가 홀수이면 가운데 1글자
- 문자열 길이가 짝수이면 가운데 2글자
### 입출력 예시
```
예시 1:
입력: s = "abcde"
출력: "c"

예시 2:
입력: s = "qwer"
출력: "we"
```

### 문제 이해
- 입력: 문자열 `s`가 주어진다.
- 해야 할 일:
  - 문자열의 길이를 기준으로 가운데 위치를 찾는다.
  - 길이가 홀수이면 가운데 문자 1개를 반환한다.
  - 길이가 짝수이면 가운데 문자 2개를 반환한다.
- 주의할 점:
  - Java 문자열 인덱스는 0부터 시작한다.
  - `substring(start, end)`에서 end 인덱스는 포함되지 않는다.
  - 짝수 길이일 때는 가운데 왼쪽 문자와 가운데 오른쪽 문자를 함께 반환해야 한다.
- 반환 형식: 가운데 글자를 담은 `String`

### 접근법
1. Solution1: substring 방식 (권장)
   - 문자열 길이의 절반 인덱스를 기준으로 홀수와 짝수 경우를 나누어 `substring()`으로 필요한 범위만 잘라 반환한다.
2. Solution2: charAt 방식
   - `charAt()`으로 직접 문자를 꺼내 문자열로 합칠 수도 있다.
   - 다만 짝수일 때 문자 2개를 직접 더해야 하므로 덜 간결하다. 

### 코드 설계
1. 실제 코드의 첫 번째 동작: 문자열 s의 길이를 구한다.
2. 초기화할 변수
   - `int length = s.length();`
   - `int mid = length / 2;`
3. 조건문 안에서 할 일
   - 길이가 홀수이면:
     - mid 위치의 문자 1개를 반환한다.
     - 범위는 `substring(mid, mid + 1)`이다.
   - 길이가 짝수이면:
     - `mid - 1`부터 `mid` 위치까지 문자 2개를 반환한다.
     - 범위는 `substring(mid - 1, mid + 1)`이다.
4. 반환 형식: 가운데 글자를 담은 문자열을 반환한다.

## 코드 구현

### Solution1: 방식 (권장)
- 핵심 포인트
  - 문자열 길이의 절반 인덱스 `mid`를 기준으로 가운데 위치를 계산한다.
  - `substring(start, end)`의 end는 포함되지 않는다는 점을 이용한다.
- 시간복잡도: O(1)
  - 문자열 길이가 최대 100으로 고정되어 있어 코딩테스트 기준에서는 상수 시간으로 볼 수 있다.
  - 일반적으로 잘라내는 문자열 길이에 비례한다고 보면 O(K)이다.
- 공간복잡도: O(1)

### Solution2: 방식
- 핵심 포인트
  - `charAt()`으로 가운데 문자를 직접 꺼낸다.
  - 홀수일 때는 문자 1개를 문자열로 변환한다.
  - 짝수일 때는 문자 2개를 문자열로 이어 붙인다.
- 시간복잡도: O(1)
- 공간복잡도: O(1)
