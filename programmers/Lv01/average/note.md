# Problem: 평균 구하기

### 문제
문제: 배열의 평균을 구하여 실수로 반환하세요.
```text
정수를 담고 있는 배열 arr의 평균값을 return하는 함수, solution을 완성하시오.
```
- 제한사항
  - arr은 길이 1 이상, 100 이하인 배열이다.
  - arr의 원소는 -10,000 이상 10,000 이하인 정수이다.

### 입력
- 정수를 담고 있는 배열 `arr`

### 출력
- 배열 `arr`의 평균값
- 반환 타입은 `double`

### 입출력 예시
```
예시 1:
입력: [1, 2, 3, 4]
출력: 2.5

예시 2:
입력: [5, 5]
출력: 5
```
