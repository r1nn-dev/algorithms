# Problem: Max and Min

### 문제 
문제: 공백으로 구분된 숫자 문자열이 주어질 때, 최솟값과 최댓값을 `"최솟값 최댓값"` 형식으로 반환하라.
```
문자열 s에는 공백으로 구분된 숫자들이 저장되어 있습니다.
str에 나타나는 숫자 중 최소값과 최대값을 찾아 이를 "(최소값) (최대값)"형태의 문자열을 반환하는 함수, solution을 완성하세요.

예를들어 s가 "1 2 3 4"라면 "1 4"를 리턴하고, "-1 -2 -3 -4"라면 "-4 -1"을 리턴하면 됩니다.
```
- 제한사항
  - s에는 둘 이상의 정수가 공백으로 구분되어 있다.

### 입력
- 공백으로 구분된 정수 문자열 `s`

### 출력
- 최솟값과 최댓값을 `"min max"` 형식의 문자열로 반환한다.

### 입출력 예시
- 문자열을 공백 기준으로 나눈 뒤, 정수로 변환하여 최솟값과 최댓값을 찾는다.
```
예시1: 
입력: "1 2 3 4"
출력: "1 4"

예시2: 
입력: "-1 -2 -3 -4"
출력: "-4 -1"
```

### 문제 이해
- 입력: 공백으로 구분된 정수 문자열 `s`
- 문자열 안의 숫자들을 분리한 뒤, 그중 최솟값과 최댓값을 찾아야 한다.
- 숫자는 음수도 포함될 수 있다.
- 문자열 비교가 아니라 반드시 정수로 변환한 뒤 비교해야 한다.
- `min`, `max`를 0으로 초기화하면 모든 숫자가 음수인 경우 잘못된 결과가 나올 수 있다.
- 반환 형식: `"min max"`

### 접근법
1. 한 번 순회하면서 min/max 갱신
    - 숫자를 하나씩 확인하면서 현재 최솟값, 최댓값과 비교하면 한 번의 순회로 답을 구할 수 있다.
2. 정렬 
    - 숫자들을 배열로 만든 뒤 정렬한다.
    - 맨 앞 = 최솟값, 맨 뒤 = 최댓값

### 코드 설계
1. 문자열 `s`를 `" "` 기준으로 나눈다.
2. 첫 번째 값을 정수로 변환해서 `min`, `max`를 초기화한다.
3. 두 번째 값부터 마지막 값까지 반복하면서:
    - 현재 값 `num`이 `min`보다 작으면 `min` 갱신
    - 현재 값 `num`이 `max`보다 크면 `max` 갱신
4. 반환 형식: `min max`


## 코드 구현

### Solution1: 반복문 방식 (권장)
- 핵심포인트
    - `split(" ")`으로 문자열을 분리하여 각 숫자 문자열이 배열에 저장된다.
    - `Integer.parseInt()`로 정수로 변환한 뒤 비교한다.
    - `min`, `max`는 첫 번째 값으로 초기화해야 음수 케이스도 안전하게 처리된다.
        - 0으로 초기화하면 음수 케이스에서 문제가 생길 수 있다.
- 시간복잡도: O(N)
  - `N`은 숫자의 개수
- 공간복잡도: O(N)
  - `split()`으로 문자열 배열을 생성한다.

### Solution2: 정렬 방식 
- 핵심 포인트
    - 숫자 문자열을 정수 배열로 변환한다.
    - `Arrays.sort()`로 오름차순 정렬한다.
    - 정렬 후 맨 앞은 최솟값, 맨 뒤는 최댓값이다.
- 시간복잡도: O(N log N)
  - 정렬 비용이 발생한다.
- 공간복잡도: O(N)
  - 정수 배열을 추가로 사용한다.
