# Problem: 두 정수 사이의 합

### 문제
문제: a와 b 사이의 모든 정수를 더한 값을 반환하세요.
```text
두 정수 a, b가 주어졌을 때 a와 b 사이에 속한 모든 정수의 합을 리턴하는 함수, solution을 완성하세요.
예를 들어 a = 3, b = 5인 경우, 3 + 4 + 5 = 12이므로 12를 리턴합니다.
```
- 제한사항
  - a와 b가 같은 경우는 둘 중 아무 수나 리턴.
  - a와 b는 -10,000,000 이상 10,000,000 이하인 정수이다.
  - a와 b의 대소관계는 정해져있지 않다.

### 입력
- 정수 a, b
### 출력 
- a와 b 사이에 있는 모든 정수의 합 
### 입출력 예시
```
예시 1:
입력: a = 3, b = 5
출력: 12 (3+4+5)

예시 2:
입력: a = 3, b = 3
출력: 3

예시 3:
입력: a = 5, b = 3
출력: 12
```


## 풀이

- 주제: 수학, 루프
- 문제 핵심: 등차수열의 합 공식 또는 루프로 계산
- 최초 풀이: 260430 
- 수정 버전: 260505

### 문제 이해
- 입력: 정수 a, b
- a와 b 사이(양 끝값 모두 포함)에 있는 모든 정수를 더한다.
- a가 항상 b보다 작다는 보장이 없으므로 먼저 작은 값을 시작값, 큰 값을 끝값으로 처리한다.
- `a == b`인 경우에는 그 값 하나만 포함되므로 그대로 반환하면 된다.
- 반환 형식: 두 정수 사이의 모든 정수 합을 long 타입으로 반환한다.

### 접근법
1. 방법1: 등차수열 합 공식 
   - a와 b 사이의 수들은 공차가 1인 등차수열이다.
   - 공식: `합 = (첫 항 + 마지막 항) * 항의 개수 / 2`
   - 반복문 없이 한 번의 계산으로 답을 구할 수 있으므로 효율적이다.
2. 방법2: 반복문 
   - 작은 값부터 큰 값까지 반복하면서 하나씩 더하는 방식
   - 직관적이지만 범위가 커질수록 반복 횟수가 많아진다.


## Solution1: 등차수열 합 공식 (권장)
### 코드 설계
1. a와 b 중 작은 값을 min, 큰 값을 max로 할당한다.
2. 항의 개수 `count = max - min + 1`
3. 등차수열 합 공식: `(min + max) * count / 2`
4. 반환 형식: 계산된 합을 long 타입으로 반환한다.
### 코드 구현
- 핵심 포인트
  - 두 정수 사이의 값들은 공차가 1인 등차수열이다.
  - `int` 오버플로우를 피하기 위해 `long`으로 변환해서 계산한다. 
- 시간복잡도: `O(1)`
- 공간복잡도: `O(1)`

## Solution2: 반복문 누적 방식
### 코드 설계 
1. a와 b 중 작은 값을 min, 큰 값을 max로 할한다.
2. 초기화할 변수
   - `long sum`
3. 반복문 안에서 할 일
   - min부터 max까지 순회하면서 값을 `sum`에 누적한다.
4. 반환 형식: 계산된 합을 `long` 타입으로 반환한다.
### 코드 구현 
- 핵심 포인트
  - 직관적인 풀이지만, 반복 횟수가 입력 범위에 비례하여 비효율적이다. 
- 시간복잡도: `O(|a - b|)`
- 공간복잡도: `O(1)`
