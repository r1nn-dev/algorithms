# Problem: 문자열 내 p와 y의 개수

링크: https://school.programmers.co.kr/learn/courses/30/lessons/12916

주제: 문자열, 카운팅

문제: 문자열 s에서 p(또는 P)와 y(또는 Y)의 개수가 같으면 true, 다르면 false를 반환하세요.

- 제한사항:
  - 문자열 s의 길이 : 50 이하의 자연수
  - 문자열 s는 알파벳으로만 이루어져 있다.
    
- 입출력 예
  
  ```
  입력: "pPoooyY"
  출력: true (p가 2개, y가 2개)
  ```

- 핵심
  - 문자열 순회, 대소문자 처리 (toLowerCase)


## 풀이
### 방법1: for문 기본
- 핵심
  - 문자열 → 소문자로 통일
  - p, y 개수 각각 카운팅
- 시간복잡도: O(n)
### 방법2: 단일 카운터
- 핵심
    - 변수 count 하나
    - 문자열 → 소문자로 통일
    - p가 나오면 +1, y가 나오면 -1
    - → count가 0이면 두 문자의 개수가 같음 = true
- 시간복잡도: O(n)
### 방법3: 대소문자 직접 비교
- 핵심
  - 변수 count 하나
  - p가 나오면 +1, y가 나오면 -1
  - → count가 0이면 두 문자의 개수가 같음 = true
- 시간복잡도: O(n)
