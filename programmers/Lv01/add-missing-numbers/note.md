# Problem: Add Missing Numbers

문제: 0부터 9까지의 숫자 중 numbers에 없는 숫자들의 합을 반환하라.
```text
0부터 9까지의 숫자 중 일부가 들어있는 정수 배열 numbers가 매개변수로 주어집니다.
numbers에서 찾을 수 없는 0부터 9까지의 숫자를 모두 찾아 더한 수를 return 하도록 solution 함수를 완성해주세요.
```
- 제한사항
  - 1 ≤ numbers의 길이 ≤ 9
  - 0 ≤ numbers의 모든 원소 ≤ 9
  - numbers의 모든 원소는 서로 다르다.

### 입력
- 정수 배열 numbers
### 출력 
- 0부터 9까지의 숫자 중 numbers에 없는 숫자들의 합
### 입출력 예시
```
예시 1:
입력: [1,2,3,4,6,7,8,0]
출력: 14 (5 + 9)

예시 2:
입력: [5,8,4,0,6,7,9]
출력: 6 (1 + 2 + 3)
```

### 문제 이해
- 입력: 정수 배열 `numbers`가 주어진다.
- numbers에는 0부터 9까지의 숫자 중 일부가 들어 있다.
- 0부터 9까지의 숫자 중 numbers에 없는 숫자들을 찾아야 한다.
- 찾은 숫자들을 모두 더한 값을 반환한다.
- 반환 형식: 없는 숫자들의 합을 int형으로 반환한다.

### 접근법
1. 방법1: 전체 합에서 numbers의 합을 빼는 방식 (권장)
   - 전체 합 45에서 numbers의 모든 원소 합을 빼면 없는 숫자들의 합만 남는다.
2. 방법2: boolean[] 플래그 배열로 존재 여부를 확인하는 방식
   - 숫자 0~9를 인덱스로 사용해 등장 여부를 표시한 뒤, 표시되지 않은 숫자만 더한다.
3. 방법3: Set<Integer>로 존재 여부를 확인하는 방식
   - numbers의 값을 Set에 넣고, 0부터 9까지 순회하면서 contains()로 존재 여부를 확인한다.

### 코드 설계
1. answer를 0부터 9까지의 전체 합인 45로 초기화한다.
2. 초기화할 변수
   - int answer = 45
   - 전체 합에서 입력 배열에 존재하는 숫자를 하나씩 제거하기 위한 변수이다.
3. 반복문 안에서 할 일
   - numbers를 처음부터 끝까지 순회한다.
   - 더하는 방식이 아니라 answer -= num으로 이미 존재하는 숫자를 제거한다.
4. 반환 형식: numbers에 없는 숫자들의 합을 int형으로 반환한다.

## 코드 구현
### Solution1: 전체 합에서 numbers의 합을 빼는 방식 (권장)
- 시간복잡도: `O(N)`
- 공간복잡도: `O(1)`

### Solution2: boolean[] 플래그 배열로 존재 여부를 확인하는 방식
- 시간복잡도: O(N + 10) → `O(N)`
- 공간복잡도: O(10) → `O(1)`
   
### Solution3: Set<Integer>로 존재 여부를 확인하는 방식
- 시간복잡도: O(N + 10) → `O(N)`
- 공간복잡도: O(10) → `O(1)`
